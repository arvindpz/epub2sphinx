# epub2sphinx

epub2sphinx is a tool to convert epub files to ReST for Sphinx.

## Usage
```
epub2sphinx [-o output_directory] [-t sphinx_theme_name] file.epub
```

# Installation
- Install pandoc
- Install epub2sphinx
  ```bash
  python setup.py install
  ```
